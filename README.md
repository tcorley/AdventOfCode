# How to Run solutions

## 2021

From project root:

```
yarn run:<mode> <day> <part>
```

where:

- mode: `data` or `sample`
- day: 1-25
- part: 1 or 2

## 2020

Navigate to a day

```zsh
$> cd 2020/day1
```

then run `node`:

```zsh
$> node .
```

Since each day has 2 problems, there will either be 2 outputs or a commented out section for part 1 or part 2.

Each prompt is linked in the solution header.
